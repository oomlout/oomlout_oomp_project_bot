## Akita  by adamjvr

- Akita  
Akita is a Beagle SiP and Lattice ECP5 FPGA based computing platform, combining the power of Linux and Hardware!  
  
Quick Specs:  
- Lattice LFE5U-85F-6BG381C FPGA 84K Logic Cells  
- Octavo Systems OSD3358-512M-BSM
