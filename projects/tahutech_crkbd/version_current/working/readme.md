## crkbd  by TahuTech  
  
(snippet of original readme)  
  
- Corne keyboard  
  
The Corne keyboard is a split keyboard with 3x6 column staggered keys  
and 3 thumb keys, based on [Helix](https://github.com/MakotoKurauchi/helix).  
Crkbd stands for Corne Keyboard.
