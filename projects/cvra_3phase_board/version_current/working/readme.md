CVRA 3 Phases Controller  
========================  
  
Features:  
---------  
   
- 3 phases  ( BLDC, DC motor, Load )  
- 10A per phase  
- Operating voltage min 48V  
- CAN interface  
- Connection for Hall sensor  
- Connection for encoder with A/B/(I) interface (5V tolerant)  
- Current measurement  
- 1x LED  
- Bus connector -> PicoBlade
