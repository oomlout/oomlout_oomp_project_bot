## Squid60  by Alasofia  
  
(snippet of original readme)  
  
- Squid60  
GH60 Compatible PCB using Hi-Tek 725 Series (Space Invaders) switches.  
  
Supports 8255C+/RT101 layouts.  
  
Thanks to Engicoder and ai03 for the footprints and PCB design guide.
